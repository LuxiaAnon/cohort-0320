Instructions

Step 1

Add an input and a button to the markup
@click on the button, you should take the input value and and use it to add a student to the student list.
Reset the input.
You are free to use whatever technique you want.
hint: innerHTML, createElement...

Step 2

Add an event handler @click either on the student list parent (the ul)
or individually on the li's when creating them. The event handler should toggle / add / remove a class to the clicked li
Create the class in the css in order to see the changes when the element is clicked. You're free to choose the properties you want to apply.
Make it a transition - You're free to choose what property you want.

Bonus

Add a delete button to the markup
Add an event handler to the button that should remove all the elements that have been selected.
(The selected elements are the elements that have a class applied to them when a user has clicked on them)
